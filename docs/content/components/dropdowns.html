--- title: Dropdowns description: | An inline menu which can be used to provide additional actions and links related to an object on the page. usage: | Use contextual menus when you need to relate a set of actions or links to an object on the page, and when space does not afford explicit links. Contextual actions are by nature 'hidden' and thus should be avoided for any critical actions. notes: | Where possible, avoid nesting contextual menus as these can be difficult to use. Creating multiple menu launchers - each with a specific goal and related set of actions - can be an effective alternative. Limiting the number of actions on an object, and pushing less important actions to a 'detail' page, can also be effective. ---
