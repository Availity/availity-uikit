UIKit

The Availity UIKit is derived from Bootstrap 3 and allows developers to create beautiful web applications for the Availity Spaces platform.

This guide will document any modifications done to the Bootstrap framework. Original Bootstrap conventions were preserved as much as possible. The Availity UIKit is available on Github and Bower.
